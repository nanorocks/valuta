za kreiranje na nova granka

- prvo vidi na koja granka si
	- git branch
- kreiraj nova granka
	- git checkout -b <imeto_na_novata_granka>

- za zakacuvanje na vekje postoecka granka
    - git checkout <imeto_na_grankata>

- za prikacuvanja na novi raboti na granka
	- dali ima promena na granka
		- git fetch --all      ( ke ti kaze dali ima promena ili ne )
	- ako ima promeni togas
		- git pull
	- da proveris dali e se vo red odis
		- git branch  --> ke ti kaze na koja granka si
		- git branch origin/<imeto_na_lokalnata_granka>
	- da gi zakacis promenite
		- git add <pateka_do_fajl_sto_prikacuvas>
	- otkako ke dodades se sledno e
		- git status  (ke ti pozeleni toa sto ti e staveno)
	- naredno sledi da komitiras
		- git commit -m "imeto na porakata" (pred sekoe ima na poraka koristis rezerviran zbor <ADD, FIX, WIP, RM> )
	- naredno treba da gi stavis promenite na granka
		- ako grankata e novo kreirana se dodava i flag
			- git push -u origin/<imeto_na_lokalnata_granka>
		- avo sportivno samo
			- git push  origin/<imeto_na_lokalnata_granka>


** Rabotis samo na develop granka, na master ne pravi push nikogas.
